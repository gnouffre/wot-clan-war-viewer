Interaction with Map

Direct Controls

Direct controls are available on map screen. They allow :

-Zoom with the control +/-

-Fullscreen with the control <-->

-Image Save (right click on map)

-Click on province for detail on it

-DblClick on province for openning The Official Wargaming province

InDirect Controls

Indirect controls allow to modifiy the appearance of the map acting on :

-Display mode

-Filters province

Display mode : Clan

Province are colored by the clan-color and the emblem is shown. If province have no Owner, no color and no emblem are show. Owner can be missing cause the province was abandonned and the owner are Civilians, or the Clan is not yet referenced as owner by wargaming, just wait next syncronization.

Display mode : Front

With a Red and blue gradient, one color is affected to each distinct front. Then this color is affected to the province acording to the front property of this province.

Display mode : Prime Time

Similar method than front, Red to Blue gradient is affected to each province acording to his prime time property.

Display mode : Revenue / Level

The revenu is the base income of the province. The level represent the infrastructure of the province, exact income depend of those values. The map is displayed with green to white gradient, the green color is stronger if the revenu or level is important. The exact revenu or level is also written on the province.

Display mode : Battles / Battles V2

During the day clan can attack a neigbourg province, or make a debarquement on landing province. On the first case they are named Attackers, on the second Competitors. The first mod Battle allow to show color clan map, with indicator of Attackers + Competitors. The second mod Battles V2 is more elaborate, it show the same value but province are in red if battles are actives, and green if battles are finished or not beginned. There is also a gradient to indicate the intensity of battles.

Display mode : Language

!!!This function is experimental and not 100% accurate !!!.

Each different language detected bring a random color on map. Each language is also represented with a flag

The first time clan succeded a debarquement, his language was analyzed. This was done with the "Clan Description". This text was submit to a language analyzer that give a estimated language.

Display mode : ELO

Similar to revenu, The map is displayed with green to white gradient, the green color is stronger if the ELO is important. The exact ELO is also written on the province.

3 elo mode are present for level VI, VIII, X.

Fine Level is also allowed, this value is given by wargaming to evaluate if a clan fight on the right front according to his ELO values.

Display mode : Province

This mod show province on map labelled by their name.

Filter : using

A large choice of filter are available on the left of the map. They are cumulative, we can use multiple selector on same time. When Filter are choosen, just click on "Filter" Button to applied the search. Majority of them are multiselect, we can choose one or more value, there is also range number value where min and max value can be positionned for the search. If u want to remove all the filters, just click on "Reset" Button.

Filter : Effect on Map

When a search was applied, the map was reactualized to show the filtered province. So with Filter and Display mode we can build hundreed of differents map.

Filter : Effect on Data

When a search was applied, the data are calculate according to the filter choice. Those data are visible in a special table.

This table can be sorted ASC DES on click on the header column, The first column, show the province name which is clickable. On click the map is automaticly positionned the province selected.

Refresh & Historic

Refresh Controls

By default the server make a synchronisation with wargaming every hour. The first button on the left of the map force the server to make a new synchronisation with the wargaming data. This action is only allowed once time per 5 minutes, if a recent synchronisation was detected, the data are only refresh by the last save.

Historic

Every night, the server execute a save of the last map. The map can show those data. Only the minimal data are saved, so some view are inactived (Battles). Some basic data are saved like level, owner of province are accurate, more complex value like ELO are not historized, so the value shown are the last one.

Exporting Map

Image mode (PNG)

The map is already rendered as an image. No direct download is possible due to CORS problem, but just make right click on map and choose "Save as" to download ur custom map

Google Earth (KML)

The map can be downloaded in Google Earth format. Just click on the Google earth icon on the map, a KML file is generated, open it with google earth application to view ur generated map in 3D
